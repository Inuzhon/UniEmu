# Термины и источники

## Термины

| Термин | Значение |
| --- | --- |
| UniEmu | Вся система: backend, frontend, runtime и scripting API. |
| Эмулятор | Виртуальный источник промышленной телеметрии. |
| Тег | Одна переменная, которую runtime может вычислить и отправить. |
| Preview | Последнее или начальное строковое представление значения тега. |
| Runtime | Quartz-based механизм вычисления и публикации данных. |
| Dispatcher | Внешняя система, принимающая мониторинг и управляющие программы. |
| Universal protocol | Протокол обмена monitoring payload и CNC-файлами. |
| CSX | C# script, используемый для вычисления тегов. |
| Scope | Область видимости `shared` или `emulator`. |
| Telemetry point | Сохраненный снимок значений тегов в момент времени. |
| System event | Событие backend/runtime, отображаемое в UI и realtime. |
| SignalR | Канал realtime-обновлений backend -> frontend. |
| Generator | Источник значения тега, вычисляющий сигнал по времени. |
| Scenario | Timeline из нескольких generator-сегментов. |
| Monaco | Редактор кода, встроенный в страницу скриптов. |
| PWA | Progressive Web App: устанавливаемая веб-версия приложения с Service Worker и manifest. |
| Service Worker | Браузерный worker, который кэширует frontend-ресурсы и обслуживает offline fallback. |
| Сценарий применения | Набросок конфигурации эмулятора под конкретный тип оборудования или проверку. |
