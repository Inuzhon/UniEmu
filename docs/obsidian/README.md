# UniEmu Obsidian Vault

Откройте папку `docs/obsidian` в Obsidian как отдельный vault. Главная точка входа: [[00 Главная]].

Документация написана обычным Markdown с Obsidian wikilinks, таблицами, Mermaid-схемами и callout-блоками. Она не требует плагинов.
